## How to use this guide

The most important sections to read are the **Overview** and the **Key mechanics**.  In addition, anything marked with your role tag is important to read.  For example, if you are a tank, you should read the **Tank** section.

### Tags

There are a handful of tags used throughout this guide to help you find the information you're looking for.

| Tag        | Description                                                     |
| :--------- | :-------------------------------------------------------------- |
| DPS        | Info for DPS players.                                           |
| Healer     | Info for healers.                                               |
| Tank       | Info for tanks.                                                 |
| Non-tanks  | Info specific to non-tank players.                              |
| @tactic    | A tactic for the previous mechanic or section.                  |
| @critical  | A critical or dangerous mechanic.                               |
| @countdown | Indicates a bossmod countdown should be used for this mechanic. |
| @tip       | A tip for the previous mechanic or section.                     |

#### Class tags

There are also tags for each class.  These are used to indicate class-specific mechanics or tactics.

<Callout>
  For authors, any text with the name of a class will automatically be color-coded to match the class color.  E.g. Paladins, warlock, MoNks, Evoker, etc.
</Callout>

### Tabs

You'll also see tabbed sections that contain additional, relevant information for the parent section. Click on each tab header to display the tabbed content.

For example:

<Tabs items={['Tab A', 'Tab B']}>
  <Tabs.Tab>Tab A content</Tabs.Tab>
  <Tabs.Tab>Tab B content</Tabs.Tab>
</Tabs>
